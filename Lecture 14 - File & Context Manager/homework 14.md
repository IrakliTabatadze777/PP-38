## 🌟 დამატებითი ამოცანა (OPTIONAL)

### faker და random — მოკლე გაცნობა

ამ ამოცანაში დაგჭირდება ორი ბიბლიოთეკა:

**`random` — შემთხვევითი რიცხვები** (Python-ში ჩაშენებული, `import random` საკმარისია):
```python
import random

random.randint(0, 100)   # მთელი რიცხვი 0-დან 100-მდე (ჩათვლით)
random.randint(50, 95)   # მთელი რიცხვი 50-დან 95-მდე
```

**`faker` — ყალბი, მაგრამ რეალისტური მონაცემები** (ინსტალაცია: `pip install faker`):
```python
from faker import Faker

fake = Faker()

fake.name()          # → "Ana Beridze"
fake.first_name()    # → "Giorgi"
fake.email()         # → "nino@example.com"
fake.address()       # → "Tbilisi, Rustaveli 42"
```

> 💡 ქართული სახელებისთვის: `Faker("ka_GE")` — `pip install faker` ჩასმის შემდეგ სცადე!
