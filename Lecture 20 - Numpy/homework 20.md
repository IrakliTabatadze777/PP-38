## ➕ 3. პროდუქტის დამატება

* რაოდენობა (stock)

ახალი პროდუქტი უნდა დაემატოს `products.csv` ფაილში.

თუ პროდუქტი არ მოიძებნა, უნდა დაიბეჭდოს შესაბამისი შეტყობინება.

## 🧾 5. მოქმედებების ლოგირება (`log.txt`)

ყველა მოქმედება უნდა ჩაიწეროს `log.txt` ფაილში.

ლოგში უნდა ინახებოდეს:

* მომხმარებლის სახელი
* მოქმედების ტიპი
* დრო (timestamp)
* დამატებითი ინფორმაცია (თუ საჭიროა)

---

# ⏱️ მინიშნება — მიმდინარე დროის მიღება Python-ში

ლოგის ჩასაწერად გამოიყენეთ `datetime` მოდული:

```python id="datetime_hint"
from datetime import datetime

# datetime.now()
# აბრუნებს მიმდინარე თარიღსა და დროს კომპიუტერის სისტემური საათიდან
current_time = datetime.now()
```

📌 **რა არის `datetime.now()`?**

* აბრუნებს **მიმდინარე თარიღსა და დროს**
* მაგალითად: `2026-06-03 14:20:11.123456`
* ანუ შეიცავს:

  * წელს
  * თვეს
  * დღეს
  * საათს
  * წუთს
  * წამს
  * მილიწამებსაც კი

---

📌 **რა არის `strftime()`?**

```python id="strftime_hint"
current_time = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
```

* `strftime()` ნიშნავს: **String Format Time**
* ის გარდაქმნის `datetime` ობიექტს **სასურველ ტექსტურ ფორმატში**
* ანუ:

  * `datetime` → string (ტექსტი)
* მაგალითად შედეგი:

```
2026-06-03 14:20:11
```

📌 ფორმატის მნიშვნელობა:

* `%Y` → წელი (2026)
* `%m` → თვე (06)
* `%d` → დღე (03)
* `%H` → საათი (24-საათიანი ფორმატი)
* `%M` → წუთი
* `%S` → წამი
